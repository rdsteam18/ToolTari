About Text Case Converter

This free online tool allows you to convert text between different letter cases:

  • UPPERCASE - Converts all letters to uppercase
  • lowercase - Converts all letters to lowercase
  • Title Case - Capitalizes the first letter of each word
  • Sentence case - Capitalizes the first letter of each sentence
  • camelCase - Removes spaces and capitalizes first letter of each word except the first
  • PascalCase - Similar to camelCase but capitalizes the first letter too
  • snake_case - Replaces spaces with underscores
  • kebab-case - Replaces spaces with hyphens

All processing happens in your browser - your text is never sent to any server.
